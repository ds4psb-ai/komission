### 2. 🔴 실행자 → 전략적 파트너

**역제안 시스템 도입:**
```
크리에이터가 "룰을 깨고 변주" 체크 → 업로드
    ↓
결과가 좋으면 → CreatorSignal 기록
    ↓
Slot → Candidate → DNA 승격 (기존 로직)
    ↓
승격 시 → 크레딧/배지/수익쉐어 자동 발생
```
