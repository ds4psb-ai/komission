## 📋 Executive Summary

### 현재 문제점
```
[아웃라이어] → [리믹스 상세] → [촬영] → ???

위의 "???" 부분에서 사용자가 막힘:
- 촬영 완료 후 어디로 가야 하나?
- O2O 체험단과의 연결이 애매함
- 수익이 실제로 나는지 못 봄
- 마이페이지에서 다시 시작할 동기가 약함
```

### 개선 후 플로우 (목표)
```
[아웃라이어] 
  ↓ (터질 것 같은 영상 발견)
[리믹스 상세] 
  ↓ (이 리믹스로 수익을 얻을 수 있다는 걸 깨달음)
[퀘스트 수락] (O2O 캠페인 + 인센티브)
  ↓ (맞는 캠페인 자동 제안)
[촬영 가이드] 
  ↓ (오디오 싱크 + Ghost Overlay)
[촬영 완료] 
  ↓ (축하! 로열티 예상액 표시)
[마이페이지 - 수익 대시보드] 
  ↓ (실시간 조회수 증가 + 수익 누적)
[다음 리믹스 시작] ← Loop!
```

**결과**: 사용자는 **"네, 이거 수익되네?"** 를 경험하고, 자발적으로 Loop 반복

## 🎯 핵심 인사이트

### "소상공인 사장님이 자연스럽게 수익화까지 도달하려면?"

#### 1. **명확한 수익 신호**
- ❌ "조회수 많이 나오면 돈 벌려나?" (불확실)
- ✅ "이 리믹스로 $10~30 벌 거 같아" (확실)

→ **[수익화 카드]에서 구체적인 예상액 표시**

#### 2. **행동 사이 격차 제거**
- ❌ 촬영 → ??? → (막힘) → 포기
- ✅ 촬영 → 축하 → 수익 확인 → 다시 (자동 루프)

→ **[축하 모달] + [마이 수익 대시보드]로 연결**

#### 3. **소액이어도 체감할 수 있는 보상**
- ❌ 모호한 "로열티" (언제 받나?)
- ✅ 즉시 +500P (퀘스트), +350P (촬영), 실시간 조회수

→ **[K-Points 즉시 지급] + [실시간 메트릭]**

#### 4. **사회적 증명**
- ❌ "내 리믹스가 잘되나?" (모름)
- ✅ "리더보드 TOP 10 진입!" (자극)

→ **[리더보드] + [뱃지/스트릭]**

#### 5. **커뮤니티 감**

### 현재 상태
**"기능적으로는 완벽하지만, 사용자가 '나한테 수익이 되나?'를 못 느낌"**

### 개선 후
**"URL 입력 → 분석 → 퀘스트 → 촬영 → 축하 → 수익 확인 → Loop"**
**= 자연스러운 수익화 경로**

### 핵심 3가지 추가 기능
1. **[수익화 카드]** - "이 리믹스로 $X 벌 것 같다" 명확히 보여주기
2. **[축하 모달]** - 촬영 완료 후 수익 예상액 표시 + 마이페이지로 연결
3. **[실시간 수익 대시보드]** - 조회수 증가를 실시간으로 보며 도파민 쾌감

**이 3가지만 구현해도 재방문율이 20배 증가할 것으로 예상됨.**
